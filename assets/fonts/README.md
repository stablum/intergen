Place a licensed Carbon Plus font file in this directory to make it the app-wide UI font.

Supported filenames:
- `CarbonPlus-Regular.ttf`
- `CarbonPlus-Regular.otf`
- `Carbon Plus Regular.ttf`
- `Carbon Plus Regular.otf`
- `CarbonPlus.ttf`
- `Carbon Plus.ttf`

If none of these files are present, the app falls back to Bevy's built-in default font.
